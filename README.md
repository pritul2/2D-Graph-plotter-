### 2D-Graph-plotter-
## Mechnism
A class Draw_conics with shapes as member functions which are called with switch case mechanism.
e.g.,
straight_line();
circle();  etc.
## Process
1) Select a curve from given options
2) Enter necessary Data
And your graph is ready
